mapViewer 0.2 für Landwirtschafts Simulator 2011
================================================
Zeigt die PDA Map auf dem ganzen Bildschirm dar.
Zeigt die Fahrzeuge, Anbaugeräte und Spieler auf der Karte an.
Spielernamen werden unter dem Symbol der Spieler angezeigt.

Angezeigt wird die Standardmap von LS
Voll Multiplayer fähig, es werden alle Spieler angezeigt

Taste(n) :
----------
Space 		=	Anzeigen/Ausblenden

Anzeige :
---------
# 			= 	Anbau gerät
* (rot) 	=	leeres Fahrzeug
* (grün) 	= 	Spieler in Fahrzeug
